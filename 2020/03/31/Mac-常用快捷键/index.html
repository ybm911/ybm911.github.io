Mac 常用快捷键

Elli0t
2020-03-31
Mac

除此之外还可以自定义很多快捷键，我设置了 option + cmd + space 全局打开设置。

系统

开机 option : 启动管理
开机 cmd + R : 打开恢复功能（Recovery）
开机 cmd + option + P + R : 重置 NVRAM
有些时候电脑会出现些小问题，重置NVRAM是你除了重新启动，尝试修复的第一选择。

cmd + option + esc : 打开强制退出程序
option + 右上角通知中心的图标 : 退出或进入勿扰模式
shift + option + 控制亮度按键 : 将亮度 1/4 变化
cmd + option + H 除了当前窗口，其他窗口全部隐藏

访达

cmd + shift + n : 新建文件夹
cmd + shift + g : 直达文件夹
cmd + o : 打开文件
cmd + ↑ : 打开包含当前文件夹的文件夹，相当于Windows里的“向上”
cmd + e : 推出所选磁盘

cmd + shift + s : 另存为
cmd + p : 打印
option + : 输出 “…”

切换

cmd + shift + Tab : 在应用程序间切换（反向）
cmd + ~ : 在同一个应用程序窗口间切换
cmd + tab 在你想唤醒出已经关闭窗口的程序时放开 tab 按下 option

浏览器

cmd + L : 光标直接跳至地址栏
cmd + Tab : 转向下一个标签栏
cmd + shift + Tab : 转向上一个标签栏

文本处理

ctrl + com + 空格键 : 调出符号窗口
shift + option + cmd + v : 移除复制的文本的格式
shift + 空格 : 输入法，拆字输入

如何给英文字母加上拼音
添加 ABC 英文拓展然后，按住 option + a、e、v、` 选择 ¯ ´ ˇ ` 四个音调
淼(miǎo)

cmd + b : 所选字体粗体（Bold）显示
fn + Delete : 向后删除
fn + ← : 跳至文章开头
fn + → : 跳至文章结尾
fn + ↑ : 向上一页
fn + ↓ : 向下一页
ctrl + a : 光标移至行首
ctrl + e : 光标移至行尾

cmd + → : 将光标移动到句子的结尾
cmd + ← : 将光标移动到句子的开头
cmd + ↑ : 将光标快速移动到整篇文本开头
cmd + ↓: 将光标快速移动到整篇文本结尾

option + ↑ : 将光标移动到段落的结尾
option + ↓ : 将光标移动到段落的开头
option + ← : 光标移至上一个单词的开头
option + → : 光标移至下一个单词的结尾

以上8个快捷键中再加上一个 shift 可以选中
