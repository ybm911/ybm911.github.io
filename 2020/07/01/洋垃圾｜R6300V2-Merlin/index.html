科学上网🔬

淘宝购买的 NETGEAR R6300V2 想给一些linux机器进行科学上网（很多物联网系统很难找到好的源）

首先安装 crackey.tar.gz 软件，然后卸载（这玩意破解安装限制）

然后安装 shadowsocks_4.2.2.tar.gz 软件，安装成功后配置一下线路，即可科学上网

挂载 USB（只支持 ext4 格式）

方法一：在 mac 下，安装 e2fsprogs

brew install e2fsprogs

插上 U盘，执行

diskutil list

卸载 U盘并将其格式化为 ext2/3/4 格式
