    上传文件

    • 先在 github 上新建一个库，然后下载下来。再把文件复制进去。
      再 git add .
      git commit -m "更新情况说明"
      git push origin master 向远程库提交代码
    • git文件 取消init初始化
      cd 到文件目录里面
      执行 rm -rf .git 即可，再执行 git remote -v 查看是否将之前仓库地址清理干净 然后指定自己的仓库地址即可

    如何在GitHub搜索到你想要的东西（淘宝购物）

    高级搜索（找到你想买的）

    options commit
    location: location:china,搜索中国区域的开发者
    followers: followers:>100,搜索超过100粉丝的开发者
    stars: stars:>100,搜索 star 超过 100 的仓库
    language: language:c++,搜索 c++ 项目仓库
    Awesome: Awesome python,搜索优秀的 python 仓库

    Explore（没事就逛逛，反正不花钱）

    除了直接搜索外，GitHub 面板也有一个 explore （勘探）选项卡，点进去就可以发现更大的世界了，比如最近有什么火起来的新项目或大佬。
