sublime

在设置中直接将 ignored_packages 置空，启用 vim 模式

Mac上sublime和vscode的vim模式不能够一直点 hjkl 来移动光标

在 terminal 输入 defaults write -g ApplePressAndHoldEnabled -bool false 就行了

安装插件

  • Block Cursor Everywhere 在 vim 模式下设置光标成小方块

设置

//    设置光标闪烁样式
"caret_style": "blink",
//    超出光标上方的额外距离
"caret_extra_top": 0,
//    超出光标下方的额外距离
"caret_extra_bottom": 0,
//    超出光标宽度
"caret_extra_width": 10,
//    一个 tab 等于4个空格
"tab_size": 4,
"translate_tabs_to_spaces": true,
